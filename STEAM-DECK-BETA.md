# FROGGLE Beta - Steam Deck Setup

Thanks for testing! This takes ~2 minutes in Desktop Mode, then you play from Gaming Mode forever.

---

## One-Time Setup (Desktop Mode)

### Step 1: Switch to Desktop Mode
- Hold the **POWER** button → **Switch to Desktop**

### Step 2: Download FROGGLE
1. Open a web browser (Firefox is pre-installed)
2. Go to: **https://github.com/preswes14/FROGGLE/releases**
3. Download the `.AppImage` file from the latest release
4. Save it somewhere easy to find (like your Desktop or Downloads)

### Step 3: Make It Executable
1. Open **Dolphin** (the file manager)
2. Find the `.AppImage` file you downloaded
3. **Right-click** it → **Properties**
4. Go to the **Permissions** tab
5. Check **"Is executable"**
6. Click **OK**

### Step 4: Add to Steam
1. Open **Steam** (desktop version)
2. Click **Games** menu → **Add a Non-Steam Game to My Library...**
3. Click **Browse...**
4. Navigate to where you saved the `.AppImage` file
5. Select it and click **Open**
6. Click **Add Selected Programs**

### Step 5: (Optional) Clean Up the Name
1. In your Steam Library, find the new entry
2. **Right-click** it → **Properties**
3. Change the name to: `FROGGLE`

### Step 6: Return to Gaming Mode
- Double-click **Return to Gaming Mode** on the desktop

---

## Playing the Game

1. Open your Steam Library
2. Find **FROGGLE** in your games list (check Non-Steam section if needed)
3. Press **Play**

---

## Controls

| Button | Action |
|--------|--------|
| **D-Pad / Left Stick** | Navigate menus and targets |
| **A** | Confirm / Select |
| **B** | Back / Cancel |
| **Y** | Toggle expand selection (for multi-target) |
| **Start** | Pause menu |
| **L1 / R1** | Switch between heroes |

## Troubleshooting

**Game won't launch:**
- Make sure you checked "Is executable" in file properties
- Try double-clicking the AppImage in Desktop Mode first to verify it works

**Controller not responding:**
- Make sure you launched from Gaming Mode, not Desktop Mode
- Try restarting the game

**Game is slow or stuttery:**
- Close other apps
- Make sure Steam Deck isn't in power-saving mode
